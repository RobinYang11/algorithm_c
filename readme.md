#### 树的存储方式
##### 一 双亲存储方法. 
 - 1. 找父节点容易O(1).
 - 2. 找子节点需要遍历O(n).
 - 3. 用数组存储所有节点 .
 - 4. 每个节点记录父节点索引.
 - 5. 核心方法以及最优时间复杂度
   - a. 获取根节点 O(n)
   - b. 按层次输出树 最好的情况O(n) 最坏的情况O(n^2)  空间复杂度O(nlogn)
